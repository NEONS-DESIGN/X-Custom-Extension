# Xカスタム拡張機能

> [!WARNING]
> 個人使用のための拡張機能なのでバグなんかの対応はいたしません。

1. 複数画像アイコンの視認性向上 (半透明背景付与)
2. 画面右下にメディア欄に遷移するためのボタン追加
3. Grok・DMボタンの削除
